# Material Analysis

This repository is a showcase of the Python lecture notebooks.

## Installation

[Python](https://www.python.org/) can be downloaded via the website. Please install [Git](https://www.git-scm.com/) to clone this repository.

When you have completed the installation of the two, please run the following command on your project root folder (the folder where this README.md is).

```powershell
py -m venv .venv
```

Then please run the following command to launch the virtual environment.

```powershell
./.venv/Scripts/Activate
```

When you activate the virtual environment, please run the following command in the environment.

```powershell
pip install -r requirements.txt
```
